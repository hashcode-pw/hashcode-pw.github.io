BootCamp Structure

BootCamp is divided into three parts of four week duration. Each part will take you through a number of skills and will conclude with a minor project with the exception of Part III which consists of developing a major project of 120 - 150 man hours of effort.

Part I Weeks I - IV

Part I focuses on the fundamental concepts of web application development.

It takes you through the introduction to developing web pages with HTML, styling them with CSS, and adding interactivity to web pages with Javascript. You will also be introduced to jQuery JavaScript toolkit.

By the end of Part I you will be able to make use of intermediate level HTML and CSS techniques for developing web pages.

Part I will also introduce you to revision control systems like Git and Mercurial.

Part II Weeks V - VIII

Part II introduces you to the various datastores like SQLite, PostgreSQL, MySQL, MondoDB, and Redis.

Following that you are introduced to backend programing with Python and Django web application development framework.

By the end of Part II, you will be able to develop web applications using Python, Django, PostgreSQL(and other datastores).

You will also continue developing skills in HTML, CSS, JavaScript, jQuery, Git, and other areas introduced in Part I.

Part III Weeks IX - XII

During Part III, you will undertake a major project utilizing all the skills mastered during Part I and II. Project will be of 120 - 150 man hours effort. Project has to be completed by everyone individually.

Advance concepts like application architecture, linux system adminstration, cloud computing, automated deployment, test driven development, etc. will be covered in parallel to the project work.

You will also be introduced to the soft skills in client communication, project management, time management, etc.
